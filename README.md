### Basic Commands

```bash
# Create a new dimension
/wdn dimension create mydimension void creative

# Teleport to a dimension
/wdn teleport dimension mydimension

# List all dimensions
/wdn dimension list

# Set up auto-teleport rules
/wdn autoteleport add on_death lobby
```
